# Windows Threat Detection

## Scenario
A suspicious activity was detected on a Windows machine. The goal was to identify the attack, analyze logs, and determine the impact.

## Tools Used
- Windows Event Viewer
- Sysmon
- PowerShell
- SIEM dashboard

## Investigation Steps
- Reviewed security and system logs
- Identified abnormal processes and logon attempts
- Correlated timestamps and IP addresses
- Checked persistence and suspicious registry keys

## Findings
The activity matched malicious behavior and showed signs of compromise.

## What I Learned
- How to analyze Windows event logs
- How attackers leave traces in system logs
- How to confirm malicious behavior
