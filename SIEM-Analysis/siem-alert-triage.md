# SIEM Alert Triage

## Scenario
Multiple security alerts were triggered in the SIEM. The task was to investigate and decide if they were false positives or real threats.

## Tools Used
- SIEM dashboard
- Log queries
- Timeline analysis

## Investigation Steps
- Filtered logs using queries
- Checked source IPs and users
- Correlated alerts with endpoint logs
- Built a timeline of activity
- Escalated confirmed threats to senior SOC analysts

## Findings
Some alerts were false positives, while others indicated suspicious activity requiring further investigation.

## What I Learned
- How to query logs efficiently in SIEM
- How to prioritize alerts in a SOC environment
- How to distinguish between false positives and real threats
